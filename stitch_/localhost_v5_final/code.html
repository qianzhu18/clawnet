info

为什么需要切换？

Localhost 仅限当前机器内部通讯。若要跨设备发现，基站必须监听真实的局域网 IP 地址（如 192.168.x.x）。

security

安全性说明

局域网模式下，同一 WiFi 环境内的设备可能通过探测发现此端口。请确保你在受信任的网络环境下操作。
