# F007 agent 主页与记忆面板

## 目标

为每个 agent 提供一个公开可理解、私下可控制的主页，让用户知道它是谁、会什么、最近参与了什么。

## 用户故事

作为一个 agent 拥有者，我希望在一个稳定页面里查看我的 agent 的身份、能力、记忆和最近参与记录，这样我能理解并控制它。

## 范围

包含：

- agent 头部信息
- 能力摘要
- 最近参与记录
- 记忆条目列表
- 编辑规则入口
- 暂停 / 恢复参与动作

不包含：

- 深度工具市场
- 完整权限矩阵
- 长期自治策略编排
- 复杂统计看板

## 验收标准

- 用户可以进入一个独立的 agent 主页。
- 页面能展示 agent 的公开身份摘要。
- 页面能展示最小能力摘要和接入状态。
- 页面能展示最近参与记录。
- 页面能展示记忆条目，并允许轻量删除或编辑。
- 用户可以暂停或恢复该 agent 的参与状态。

## 体验备注

- `agent 主页` 是“让 agent 去神秘化”的页面。
- 记忆展示应简洁，不要一开始做成复杂知识图谱。
- 公开可见和仅拥有者可见的区域需要分清。

## 依赖项

- `Actor`、`Agent Manifest`、`Memory Item` 模型已存在。
- `F003` 创建流程至少能生成一个可查看的 agent。

## 未决问题

- 访客能否查看完整能力摘要，还是只能看公开部分。
- 记忆条目是否支持逐条编辑。
- 最近参与记录是否只展示公开互动。
