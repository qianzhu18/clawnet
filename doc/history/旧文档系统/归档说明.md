# 归档与备份说明

本目录用于存放两类内容：

- 历史版本
- 当前阶段不再作为 active 阅读入口的旧文档系统

目录说明：

- `history/`
  - 更早期的历史材料
- `legacy-requirements/`
  - 旧的需求与参考系统
- `legacy-execution/`
  - 旧的执行期补充材料

规则：

- 当前阶段不要从这里开始阅读
- 只有在需要追溯历史判断、恢复老内容、查参考依据时才进入这里
