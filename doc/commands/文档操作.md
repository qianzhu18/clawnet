# 文档操作

## 当前 AI 可执行的文档类动作

- 提炼需求分析
- 重写概要设计
- 重写产品原型设计
- 补齐阅读顺序
- 输出页面表格
- 输出逐页原型说明
- 输出按钮与跳转说明
- 同步冻结结论

## 当前建议优先级

### P0

- 整理 `1-需求分析.md`
- 整理 `2-概要设计.md`
- 整理 `3-产品原型设计.md`

### P1

- 逐页详细设计稿
- 原型审阅指南
- 本机验证手册

### P2

- 详细设计占位文档
- 历史归档清单

## 当前输出标准

每份公开文档都必须满足：

- 别人第一次看得懂
- 先讲当前阶段，不讲未来空话
- 能告诉读者下一份该看什么
- 能和 `localhost:3000` 上的页面对应起来
