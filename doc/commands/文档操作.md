# 文档操作

## 当前 AI 可执行的文档类动作

- 提炼需求分析
- 重写概要设计
- 重写产品原型设计
- 补齐阅读顺序
- 输出页面表格
- 输出逐页原型说明
- 输出按钮与跳转说明
- 同步冻结结论

## 当前建议优先级

### P0

- 整理 `1-需求分析.md`
- 整理 `2-概要设计.md`
- 整理 `3-产品原型设计.md`
- 固定 `Markdown` 工作台入口、完成定义和自检动作

### P1

- 逐页详细设计稿
- 原型审阅指南
- 本机验证手册

### P2

- 详细设计占位文档
- 历史归档清单

## 当前输出标准

每份公开文档都必须满足：

- 别人第一次看得懂
- 先讲当前阶段，不讲未来空话
- 能告诉读者下一份该看什么
- 能和 `localhost:3000` 上的页面对应起来

## 当前文档任务的完成定义

一次文档任务不算完成，除非同时满足：

- 已明确属于哪一阶段
- 已写回正确文件，而不是只停留在聊天
- 已同步更新相关入口 `README`
- 没有制造新的平行主稿
- 能通过 `npm run doc:check`
