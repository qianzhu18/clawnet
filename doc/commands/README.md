# commands

这里不是 Shell 命令手册，而是“你可以直接让 AI 执行什么动作”的说明区。

当前按 5 类整理：

- `文件操作.md`
- `文档操作.md`
- `测试操作.md`
- `编码操作.md`
- `附加要求.md`

## 你可以直接对 AI 说的话

### 文件操作

- `梳理当前文档目录并给出阅读顺序`
- `把重复原型文档合并成 3 份 active`
- `把旧文档归档到 history`
- `把文档目录改成更适合公开阅读的结构`

### 文档操作

- `把当前需求整理成一份对外可读的需求分析`
- `把概要设计收成一份主文档`
- `把原型拆解收成逐页审阅文档`
- `把最近讨论结论同步到标准文档`

### 测试操作

- `启动 3000 端口让我手工审核原型`
- `跑 build 和 lint`
- `给我一个本机逐页验证顺序`
- `生成当前原型的可视化回归入口`

### 编码操作

- `把首页改得更像产品页`
- `补齐基站 mock 数据`
- `把 AI 身份显示成头像角标`
- `把某个页面改到能直接展示给设计师`

### 附加要求

- `全部改成中文对外语气`
- `不要让我看到内部术语`
- `文档和 3000 端口页面一起同步`
- `不要做超出当前阶段的详细设计`
