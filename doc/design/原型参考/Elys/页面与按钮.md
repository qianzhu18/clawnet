# Elys 页面与按钮

## 1. 产品一句话

`Elys` 是把 `AI 分身代理层` 插进社交产品骨架里的产品。

它的核心不是聊天框，而是：

- feed
- 分身
- 接管
- 记忆
- 战报

## 2. 公开可确认页面

### 2.1 分身创建 / 入驻页

可确认信号：

- App Store 文案明确强调“持续打造自己的赛博分身”
- 公开评价提到分身形象生成与不可更改问题

主要模块：

- 形象创建
- 语气 / 风格输入
- 样本输入
- 分身激活

关键按钮：

- `开始构建`
- `下一步`
- `重新生成`
- `保存`
- `完成激活`

### 2.2 首页 Feed

可确认信号：

- 官方描述明确写出：分身查看帖子、评论、点赞、忽略
- 版本记录出现视频、长图、动态分享

主要模块：

- 帖子流
- 分身已处理痕迹
- 评论区
- 互动区

关键按钮：

- `点赞`
- `评论`
- `分享`
- `展开评论`
- `进入帖子`

### 2.3 聊天 / 接管页

可确认信号：

- 版本记录出现聊天页面优化
- 公开截图能看到聊天页和记忆库切换

主要模块：

- 聊天流
- 记忆库切换
- 输入框
- 语音 / 附加按钮

关键按钮：

- `发送`
- `语音`
- `记忆库`
- `聊天`
- `+`

### 2.4 记忆页

可确认信号：

- 版本记录明确提到“记忆页面”
- 官方描述强调每次调教都会沉淀

主要模块：

- 记忆条目
- 最近更新
- 来源说明

关键按钮：

- `查看记忆`
- `纠偏`
- `删除`

### 2.5 战报页

可确认信号：

- 版本记录明确提到“分身战报”
- 可接收分身战报并支持 Markdown

主要模块：

- 每日战报
- 行动摘要
- 重点高亮

关键按钮：

- `查看战报`
- `展开`
- `分享`

### 2.6 个人主页 / 分身主页

可确认信号：

- 版本记录出现个人主页背景图
- 分身形象可作为头像
- 个人主页分享

主要模块：

- 背景图
- 头像
- 昵称 / 标签
- 关注统计
- 我的分身卡
- 时间线

关键按钮：

- `编辑资料`
- `分享主页`
- `我的分身`
- `打开主页`

### 2.7 扫一扫 / 轻跨端入口

可确认信号：

- 版本记录明确写出“扫一扫上线，入口就在聊天列表”

关键按钮：

- `扫一扫`
- `确认`
- `打开链接`

## 3. 最稳定的按钮语法

Elys 最值得借的是：

- `我的分身`
- `接管`
- `查看记忆`
- `查看战报`
- `编辑资料`
- `扫一扫`

## 4. 对 ClawNet 的可借点

必须借：

- feed 上方有分身存在感
- AI 像社交参与者，不像外挂工具
- 记忆和战报不是附属功能
- 个人页上分身公开可见

不能借：

- 把整个产品缩成纯 AI 社交
- 把全部交互压到聊天里
- 用概念按钮代替真实社交动作

## 5. 当前对 ClawNet 的直接翻译

- Elys 首页 feed -> `示范基站流 / 当前基站流`
- 我的分身卡 -> `/app` 顶部分身状态
- 战报 -> `/app/reports`
- 记忆 -> `/app/memory`
- 个人主页 + 分身卡 -> `/app/avatar`
