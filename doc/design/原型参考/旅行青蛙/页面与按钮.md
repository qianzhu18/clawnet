# 旅行青蛙 页面与按钮

## 1. 产品一句话

`旅行青蛙` 不是社交产品，但它把“低控制、高期待、带回物”这件事做得极稳定。

它最关键的循环是：

- 准备
- 放手
- 等待
- 收到照片 / 礼物 / 明信片

## 2. 核心页面

### 2.1 庭院页

页面目标：

- 承担日常进入后的第一眼

主要模块：

- 三叶草
- 来信提示
- 商店入口
- 房间入口

关键按钮：

- `收三叶草`
- `看来信`
- `去商店`
- `进屋`

### 2.2 房间页

页面目标：

- 展示青蛙在家状态与准备空间

主要模块：

- 房间状态
- 行李位
- 物品位

关键按钮：

- `准备行李`
- `查看物品`
- `切换食物`

### 2.3 商店页

页面目标：

- 在出发前给用户一次准备权

主要模块：

- 食物
- 道具
- 护身符

关键按钮：

- `购买`
- `放入背包`
- `替换`

### 2.4 外出中 / 回信页

页面目标：

- 用“看不见它的时候”建立关系张力

主要模块：

- 外出状态
- 明信片
- 旅途照片

关键按钮：

- `查看明信片`
- `收藏照片`
- `继续等待`

### 2.5 相册 / 纪念品页

页面目标：

- 把带回来的东西沉淀成关系历史

主要模块：

- 相册
- 纪念品
- 地点收集

关键按钮：

- `查看相册`
- `翻页`
- `查看纪念品`

## 3. 最稳定的按钮语法

- `准备`
- `出发`
- `等待`
- `查看回信`
- `查看纪念品`

## 4. 对 ClawNet 的可借点

必须借：

- 先准备，再放手
- 离屏行动有状态
- 回来必须带回可见证据
- 证据要能沉淀

不能借：

- 纯放置心智
- 纯随机盲盒
- 没有接管点

## 5. 当前对 ClawNet 的直接翻译

- 庭院页 -> `首页中的等待感 / 基站状态`
- 外出中 -> `分身正在外部活动`
- 明信片 / 礼物 -> `战报 / 记忆 / 带回帖子`
- 相册 -> `回执沉淀页`
