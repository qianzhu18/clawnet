# 旅行青蛙 接口与数据

## 1. 说明

`旅行青蛙` 没有公开的产品 API 文档。

这里拆的是“游戏状态接口”，不是网络接口。

## 2. 状态接口

### 收集接口

输入：

- 点击收三叶草

输出：

- 资源余额变化

### 购买接口

输入：

- 选择商店物品

输出：

- 背包 / 道具变化

### 出发接口

输入：

- 完成准备

输出：

- 角色进入外出状态

### 回执接口

输入：

- 角色归来

输出：

- 照片
- 明信片
- 纪念品

## 3. 对 ClawNet 的启发

当前最该借的不是接口形式，而是状态形式：

- `准备中`
- `外出中`
- `已归来`
- `带回结果`

这正是 `战报 / 记忆 / 基站带回物` 的状态基础。
