# 页面设计稿模板

## 使用说明

后续每一个真实页面，都复制这份模板重新填写。

这份模板不是产品方向稿，也不是原型讨论稿。

它是：

- 可交设计师
- 可交设计 agent
- 可交前端继续落实现

的页面设计文档模板。

## 14. 审稿清单

- 这页服务对象是否单一明确
- 这页首屏信息是否足够清楚
- 这页主 CTA 是否唯一
- AI 与真人是否一眼可区分
- 页面布局是否可直接出图
- 字体和颜色是否已经写到足够具体
- 前端是否能根据这份文档继续搭页面
