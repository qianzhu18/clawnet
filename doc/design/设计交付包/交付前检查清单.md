# 交付前检查清单

这份清单用于检查：

- 你准备发给外部设计师的那几份文档，是否已经可以脱离仓库单独存在

## 必检项

### 1. 独立性

- 这份文档是否不依赖其他仓库文件才能理解
- 这份文档是否不依赖代码截图或源码阅读
- 这份文档是否不要求对方先看内部讨论记录

### 2. 页面背景

- 是否写明页面属于哪条链路
- 是否写明服务哪类用户
- 是否写明页面唯一主任务

### 3. 页面结构

- 是否写明首屏信息
- 是否写明模块顺序
- 是否写明布局结构

### 4. 视觉规格

- 是否写明字体
- 是否写明颜色
- 是否写明间距
- 是否写明组件尺寸

### 5. 交互规格

- 是否写明所有主按钮
- 是否写明按钮点击结果
- 是否写明默认态、异常态、成功态

### 6. 交付可用性

- 设计师是否可以不追问就开始出图
- 前端是否可以不猜测就开始实现
- 这份文档是否能单独转发给外部协作者
